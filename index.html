Power Conversion

Power (mW) = 10 ^ (dBm / 10)

Power (W) = mW / 1000

Course Width

Course Width = 2 × tan⁻¹(106.68 / Runway Length)

Distance → DDM

DDM % = 0.145 × Distance (m)

DDM → Current

Current (µA) = (150 / 15.5) × DDM %

SBO Required

SBO Req = SBO Present + Mul.Factor × log₁₀(W_measured / W_required)

SBO Narrow Alarm

SBO Narrow = 20 × log₁₀(1 / (1 − Alarm% / 100)) + SBO Present

SBO Wide Alarm

SBO Wide = 20 × log₁₀(1 / (1 + Alarm% / 100)) + SBO Present

GP DDM & Current

DDM % = (8.75 / 0.36) × (Angle Assigned − Angle Entered)

Current (µA) = (75 / 8.75) × DDM %

Antenna DDM Formulas

Lower A1 = (W_m / W_r × M_DDM) × 0.25

Middle A2 = (W_m / W_r) × M_DDM

Upper A3 90/150 = (W_m / W_r × M_DDM) × 0.00125

GP Alarm Formulas

Narrow A2 = M_DDM / (1 − 0.01 × Alarm %)

Wide A2 = M_DDM / (1 + 0.01 × Alarm %)

Clearance Current

Clearance = Present Power + 20 × log₁₀(160 / Min Current MKA)
